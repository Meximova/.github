# Meximova — Logistics & Transport Technology

Welcome to the official GitHub organization for Meximova. We design, build, and deploy modern software solutions to optimize cargo transportation, fleet compliance, and supply chain operations.

---

### Overview
At Meximova, we combine operational expertise in heavy transport with robust digital tools to deliver real-time visibility, security, and efficiency across all logistics routes.

* Internal Documentation: [Meximova Knowledge Base on Notion](https://notion.so)
* Industry: Logistics, Supply Chain & Cargo Transport

---

### Tech Stack

We leverage a modern and scalable stack to power our mobile and web ecosystem:

| Layer | Technology | Usage |
| :--- | :--- | :--- |
| Mobile App | React Native | Driver compliance app and mobile tools |
| Backend API | Laravel | Core REST APIs, business logic, and fleet management |
| Database & Infrastructure | PostgreSQL / Cloudflare | Secure data persistence, tunneling, and network safety |

---

### Key Projects & Modules
* Driver Compliance & Logistics App: Built with React Native, focusing on route management, driver safety, and operational checklists.
* Fleet Management Backend: Powered by Laravel, handling APIs, authentication, dispatching logic, and fleet tracking integration.

### Developer Resources & Guidelines
For team members and contributors:
1. Architecture & Docs: Check our Notion Workspace for onboarding guides, API specifications, and system architecture diagrams.
2. Git Workflow: 
   * Always create feature branches from develop or main (e.g., feature/driver-auth, fix/route-sync).
   * Keep commits clean and descriptive.
   * Open a Pull Request (PR) and request a code review before merging.
